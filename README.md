# AI-Powered Website-Grounded Question Answering Chatbot

## Objective
A production-grade AI chatbot that answer questions strictly grounded in a website's content.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Set up environment variables in `.env`.
3. Run the app:
   ```bash
   streamlit run app.py
   ```
